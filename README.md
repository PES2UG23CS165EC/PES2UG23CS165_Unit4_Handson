# Agentic AI System: Multimodal, Workflows & Evaluation

## Overview
This project is a comprehensive implementation of an Agentic AI System built across three major components:
1. Multimodal Models
2. Agentic Workflows
3. Evaluation & Data Ethics

The system demonstrates how modern AI:
- Handles multiple data types (text, images)
- Uses autonomous agents
- Evaluates and improves outputs
- Follows ethical AI practices

---

## Objectives
- Build multimodal AI models
- Design agent-based workflows
- Implement RAG systems
- Evaluate outputs using DeepEval
- Apply AI ethics principles

---

## Part 1: Multimodal Models
- Handles text and image inputs
- Demonstrates multimodal AI capabilities

---

## Part 2: Agentic Workflows
- Multi-agent system using CrewAI
- RAG pipeline using FAISS + LangChain
- Agents collaborate to answer, evaluate, and improve responses

---

## Part 3: Evaluation & Data Ethics
- Uses DeepEval metrics:
  - Faithfulness
  - Answer Relevancy
- Focus on responsible AI and bias reduction

---

## Tech Stack
- Python
